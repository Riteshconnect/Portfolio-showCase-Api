# Portfolio ShowCase API

A backend API for my personal portfolio built with **Node.js + TypeScript + Express**.  
Features: authentication (JWT), file uploads (Multer), real-time events (Socket.io), and ready-to-deploy configuration.

## Demo
- Live API (status):(https://portfolio-showcase-api-1.onrender.com)

## Tech stack
- Node.js, TypeScript
- Express.js
- JWT, Multer, Socket.io
- Database: MongoDB / MySQL 
- Testing: Jest (configured)
